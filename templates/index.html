Bienvenido!!

Alumno, Debe completar los siguientes endpoints en el backend:

[GET] /gastos --> HTML para mostrar todo los gastos realizados

[GET] /gastos/agregar --> HTML con el formulario para agregar un gasto a una categoria

[POST] /gastos/agregar --> ingresar nuevo gasto proveniente del formulario HTML

[GET] /gastos/[categoria] --> mostrar todos los gastos de un categoría específica en formato JSON"
